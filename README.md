## 🚀 Deployment

This site is deployed on **Vercel** and automatically deploys from the `main` branch.

### Manual Deploy

```bash
# Install Vercel CLI if needed
npm install -g vercel

# Deploy to production
vercel --prod
```

### Automatic Deployment

Push to `main` branch to trigger automatic deployment via GitHub integration.

## 🔧 Development

### Running Locally

```bash
# Start a local server
python3 -m http.server 8080

# Or using Node.js
npx serve
```

Then visit `http://localhost:8080`
